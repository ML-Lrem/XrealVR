# XrealVR
A VR180 video player for Xreal Air base on Xreal IMU API

软件基于Xreal Air的IMU API：
https://github.com/MSmithDev/AirAPI_Windows

一、播放
1. 连接你的Xreal Air设备
2. 选择一个符合要求的VR视频。【注意，仅支持180度VR视频】
3. 播放

二、控制
1. 快进后退：方向键
2. 暂停：空格
3. 调整视频纵深：W/S
4. 调整视差角度：A/D
